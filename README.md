## Introduction

## Installation

:brain: This repository is based on PyTorch 0.4.1.

## Usage

1. Clone the repository: https://github.com/zydlsd/SemiCTrans.git
2. Put the data in src/data/Brats2020Train.
3. Train the model.
4. Test the model.
